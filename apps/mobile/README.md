## Overview

Mobile application for PagoPy payment and electronic invoicing system for Paraguayan SMEs.

**Stack**: Angular 17+, Ionic 7, Capacitor 5

## Quick Start

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev:mobile      # http://localhost:8100

# Build for production
pnpm build:mobile

# Native platforms
cd apps/mobile
npx cap add android
npx cap add ios
npx cap sync
```
